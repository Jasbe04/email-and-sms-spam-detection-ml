# Spam Detection System (Email & SMS)

A Machine Learning-based Spam Detection System that automatically classifies emails and SMS messages as **spam** or **ham (not spam)**. The model uses text processing and supervised learning algorithms to accurately detect unwanted or malicious messages, helping users maintain a clean and safe inbox.

---

## Features

- Classifies both emails and SMS messages.  
- Detects spam with high accuracy.  
- Text preprocessing included: tokenization, stopword removal, and vectorization.  
- Easy to integrate into applications for real-time spam detection.  
- Handles large volumes of messages efficiently.  

---

## Technologies Used

- Python  
- Scikit-learn  
- Pandas & NumPy

## Dataset

The model is trained on a labeled dataset containing emails and SMS messages categorized as **spam** or **ham**.  
Dataset link: https://www.kaggle.com/datasets/uciml/sms-spam-collection-dataset

- **Precision:** 98%
